# Grand Theft Auto Inspired Open-World Prototype

This repository is a starter template for building an open-world sandbox game inspired by classic crime-action games.

## Features
- Vehicle controller
- NPC AI system
- Open-world map loader
- Mission scripting
- Inventory system
- Save/load support

## Stack
- Unity / Unreal / Godot compatible structure
- JSON mission files
- Modular assets folder

## Repository Structure

```txt
/assets
/scripts
/maps
/audio
/ui
/docs
```

## Example Mission JSON

```json
{
  "mission": "Midnight Run",
  "objective": "Drive to the checkpoint",
  "reward": 5000
}
```
